Ready-to-use plugin generator derived from Nuclear Taxi's Endless Sky Generator.
Modified by 1010todd

To use: go to the plugin folder and run main.py

1 Sample Galaxy is already provided, if you don't want to run the script.
A wormhole to access will be placed in a system close to Sag* A (the black hole)

This is a plugin generator intended for players to find new galaxies and make Endless Sky -practically- Endless
